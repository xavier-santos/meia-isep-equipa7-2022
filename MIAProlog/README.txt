Apenas consulte o ficheiro 'Sistema_Pericial.pl'
Antes/Depois se estiver a usar a aplicação desktop do SWI-Prolog, corra o seguinte comando trocando o X pelo caminho para a pasta do ficheiro anterior.

working_directory(CWD,X).

Exemplo:
working_directory(CWD,'home/usr/prolog').

Caso esteja a usar a versão cli do Prolog basta apenas executar o comando na mesma pasta do ficheiro.
